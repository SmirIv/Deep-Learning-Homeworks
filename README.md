# Deep-Learning-Homeworks

В этом репозитории лежат мои домашние задания по курсу «Глубокое обучение».

Каждое ДЗ лежит в собственной ветке в виде одного или нескольких ноутбуков.

## Список заданий

- **HW-01_tensor-ops-and-gradient-descent** — Операции над тензорами, градиентный спуск на NumPy.
- **HW-02_training-tricks** — Стабилизация обучения и регуляризация.
- **HW-03_finetuning-augmentation** — Классификация изображений. Дообучение. Аугментация.
- **HW-04_segmentation** — Сегментация дрожжевых клеток.
- **HW-05_detection** — Детекция дрожжевых клеток.
- **HW-06_protein-folds** — Структурная классификация белков.
- **HW-07_tiny-stories-and-translation** — TinyStories: генерация текстов, Multi30k: перевод с немецкого на английский.
- **HW-09_gan** — GAN для датасета CelebA.
